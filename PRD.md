# 日程提醒小程序 — 需求文档

> **当前版本**：v0.2（P0-1 时间选择 + P0-2 提醒推送前端 + P0-3 编辑 + P0-4 已完成标记）
> **最后更新**：2026-04-27

---

## 一、产品定位

一款轻量级微信小程序，帮助用户快速创建、查看和管理日常日程，核心路径极简：**打开 → 添加 → 查看 → 提醒**。

## 四、提醒推送接入指南（P0-2b）

### 前置条件

1. **开通云开发**：在微信开发者工具中点击"云开发"按钮，创建环境
2. **申请订阅消息模板**：
   - 登录 mp.weixin.qq.com → 功能 → 订阅消息 → 公共模板库
   - 搜索"日程提醒"相关模板，选用
   - 记录模板 ID（形如 `xxxxxxxxxx`）
3. **替换模板 ID**：将 `pages/index/index.js` 中的 `YOUR_TEMPLATE_ID` 替换为实际模板 ID

### 云函数示例

```
cloudfunctions/remind/
  index.js     — 查询即将到期的日程，调用 subscribeMessage.send()
  config.json  — 定时触发器配置（每分钟执行）
```

### 限制

- 用户每次授权只能发一条通知（一次性授权）
- 需要用户主动点击授权弹窗
- 小程序进入后台后无法主动触发，必须依赖云函数
